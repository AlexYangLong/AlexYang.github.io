什么是Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

在安装Hexo前需要确保电脑已经安装以下程序 :

  • Node.js
  • Git

这两个程序的安装很简单，windows用户可以使用安装程序进行安装，简单高效。
