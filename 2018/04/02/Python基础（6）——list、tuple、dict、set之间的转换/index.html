list、tuple、dict与set之间的转换

list和tuple是有序的，dict和set是无序的。

list和dict是可变的，tuple和set是“不可变的”。

dict具有极快的查找速度，不会因为元素个数的增多而变慢，相应的，占用的空间大；

而list占用的空间较小，但是查找速度会随着元素个数的增多而变慢。
